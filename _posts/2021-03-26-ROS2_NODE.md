COMMANDS TO USE:
- ros2 run <package_name> <node>
- ros2 pkg list
- ros2 pkg executables


FOR EXAMPLE

- ros2 run semantic_SLAM ekf_publisher.


then, it executes the node.

이전에는 익세큐트 하나에 노드 하나였는데, 로스2부터는 익세큐트에 여러 노드가 가능하다.
