# Abstract Data type
1. what is ADT(Abstract Data Type)
  - representation of data
    - **int, float, double ...**
  - operation on data.
    - **+,-,*,/,%,++,--**

### Example

- List -> 8,3,9,4,6,10,12

- Data :
  1. space for storing element(Array, Linked list)
  2. capacity
  3. size
- operation :
  - add(x)
  - remove()
  - search(key)
  - insert(key)
  - set(index, key)
  - replace(index, key)
  - get(index)
  - contain(key)
